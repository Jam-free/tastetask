## 🔧 购买后配置

### 获取服务器信息

1. 登录阿里云控制台
2. 点击"轻量应用服务器"
3. 找到你购买的服务器
4. 记录信息：
   - **公网IP**：类似 `47.96.123.45`
   - **用户名**：`root`
   - **密码**：你设置的密码

### 配置防火墙（重要！）

1. 点击服务器名称
2. 找到"防火墙"
3. 点击"添加规则"
4. 添加：
   - 端口：`3000`
   - 协议：`TCP`
   - 策略：`允许`
   - 备注：`Taste-Task应用`

**不配置这个无法访问！**

## 🚀 部署应用

### 第1步：连接服务器

在你的电脑终端（Mac）或 PowerShell/CMD（Windows）执行：

```bash
ssh root@你的公网IP
# 例如：ssh root@47.96.123.45
```

输入密码（看不见输入，正常）。

### 第2步：安装Node.js

```bash
# 安装Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node -v
npm -v
```

### 第3步：安装PM2

```bash
sudo npm install -g pm2
```

### 第4步：上传代码

**在本地电脑终端**执行（不是服务器）：

```bash
scp -r ~/Desktop/Claude\ Code/tastetask root@你的公网IP:/root/
```

### 第5步：安装依赖并启动

**在服务器上执行**：

```bash
cd /root/tastetask
npm install
node init-db.js
pm2 start server.js --name tastetask
pm2 save
pm2 startup
```

### 第6步：验证

```bash
pm2 status
# 应该显示：online
```

## ❓ 常见问题

### Q1: 找不到新用户价格？
A:
- 确保是阿里云新用户（没买过ECS）
- 找"新用户专享"标签
- 或联系客服询问活动

### Q2: 轻量服务器和ECS选哪个？
A:
- **新手**：选轻量应用服务器（简单）
- **高手**：选ECS（灵活）
- 功能上差不多，轻量服务器够用

### Q3: 1核1GB够用吗？
A:
- 个人测试：够用
- 小团队（<10人）：建议1核2GB
- 流量大：建议2核4GB

### Q4: 防火墙和端口怎么配置？
A:
- 控制台 → 轻量应用服务器
- 点击你的服务器
- 防火墙 → 添加规则
- 端口3000，协议TCP，允许

### Q5: 第二年续费多少钱？
A:
- 第一年：60-96元（新用户）
- 第二年：约100-150元
- 可以重新注册账号享受新用户价

### Q6: 数据会丢吗？
A:
- 不会，数据在服务器上
- 但建议定期备份数据库
