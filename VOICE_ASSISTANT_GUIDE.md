# 语音助手调用功能说明

## 功能介绍

在每张闪卡上添加了一个🎤按钮，点击后可以尝试唤醒手机上的语音助手，并自动询问当前卡片上的问题。

## 支持的平台

### 1. 华为鸿蒙系统 (HarmonyOS 6.0+)
- **语音助手**：小艺
- **调用方式**：
  - `hwvrcs://cmd?action=voice&query=...`
  - `huawei://voiceassistant?text=...`
  - `hwvrcs://voiceassistant?query=...`
- **检测关键词**：huawei, honor
- **提示信息**：正在唤醒华为小艺语音助手...

### 2. 小米设备
- **语音助手**：小爱同学
- **调用方式**：`miui://vlassistant/search?query=...`
- **检测关键词**：mi, redmi, xiaomi
- **提示信息**：正在唤醒小米小爱同学...

### 3. OPPO 设备
- **语音助手**：小布助手
- **调用方式**：`oppokit://voiceassistant/search?query=...`
- **检测关键词**：oppo
- **提示信息**：正在唤醒OPPO小布助手...

### 4. VIVO 设备
- **语音助手**：Jovi
- **调用方式**：`vivovoice://search?query=...`
- **检测关键词**：vivo
- **提示信息**：正在唤醒VIVO Jovi...

### 5. iOS 设备
- **语音助手**：Siri
- **调用方式**：`siri://...`
- **检测关键词**：iphone, ipad
- **提示信息**：尝试唤醒Siri（可能需要在设置中允许）...

### 6. 其他 Android 设备
- **备选方案**：使用 Web Speech API
- **提示信息**：已启动语音识别，请说出问题...

### 7. PC 浏览器
- **备选方案**：提示用户手动询问
- **提示信息**：请手动唤醒语音助手并询问：[问题]

## 使用方法

1. **打开闪卡练习**
2. **查看问题**
3. **点击🎤按钮**
4. **系统自动尝试唤醒语音助手**

## 测试建议

### 华为鸿蒙设备测试

#### 测试环境
- 设备：华为/荣耀手机或平板
- 系统：HarmonyOS 6.0+
- 浏览器：华为浏览器、微信内置浏览器、Chrome

#### 测试步骤
1. 打开网站
2. 进入闪卡练习
3. 点击任意卡片的🎤按钮
4. 观察是否成功唤醒小艺
5. 检查小艺是否自动询问问题

#### 预期结果
- ✅ 成功唤醒小艺语音助手
- ✅ 小艺自动说出问题
- ⚠️ 部分场景可能需要手动授权

#### 常见问题
- **Q: 点击后没有反应？**
  - A: 可能需要在浏览器设置中允许跳转到外部应用

- **Q: 提示"无法打开链接"？**
  - A: 尝试使用华为浏览器访问，而不是第三方浏览器

- **Q: 小艺被唤醒但没询问问题？**
  - A: 这是 HarmonyOS 的限制，需要手动说出问题

## 隐私说明

- 语音助手调用由系统处理，本网站不收集语音数据
- 仅通过 URL Scheme 触发系统功能
- 不涉及任何语音录音或传输

## 开发者注意事项

1. **HTTPS 要求**：某些平台可能需要 HTTPS 环境
2. **用户授权**：首次使用可能需要用户授权
3. **系统限制**：不同系统的支持程度不同
4. **浏览器兼容**：建议使用系统自带浏览器
5. **版本差异**：新系统版本可能改变调用方式
