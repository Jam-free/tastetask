### 第六步：获取服务器信息

#### 查看服务器信息
1. 登录华为云控制台
2. 点击"弹性云服务器 ECS"
3. 找到你刚买的服务器
4. 记录以下信息：
   - **弹性公网IP**：类似 `123.45.67.89`
   - **用户名**：`root`
   - **密码**：你设置的密码

#### 测试连接
在你的电脑终端（Mac）或命令提示符（Windows）执行：

```bash
ssh root@你的公网IP
# 例如：ssh root@123.45.67.89
```

输入密码（看不见输入，正常）。

如果连接成功，就说明服务器OK了！

## 🔧 购买后配置

### 配置安全组（必须）

1. 在华为云控制台，找到你的服务器
2. 点击"安全组"
3. 点击"配置规则"
4. 添加入方向规则：
   - 端口：3000
   - 协议：TCP
   - 源地址：0.0.0.0/0
   - 描述：Taste-Task应用

这样才能访问3000端口！

## 🚀 部署应用

### 连接服务器
```bash
ssh root@你的公网IP
```

### 安装Node.js
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证
node -v
npm -v
```

### 安装PM2
```bash
sudo npm install -g pm2
```

### 上传代码（在本地电脑执行）

**方法A：使用scp（推荐）**
```bash
scp -r ~/Desktop/Claude\ Code/tastetask root@你的公网IP:/root/
```

**方法B：使用Git**
```bash
# 如果你的代码在Git上
ssh root@你的公网IP
cd /root
git clone 你的仓库地址
```

### 安装依赖并启动（在服务器上执行）
```bash
cd /root/tastetask
npm install
node init-db.js
pm2 start server.js --name tastetask
pm2 startup
pm2 save
```

### 验证
```bash
pm2 status
# 应该显示：online
```

## ❓ 常见问题

### Q1: 找不到9元活动？
A: 活动经常变化，可能需要：
- 多找几个页面看看
- 问华为云客服
- 看看其他新用户活动

### Q2: 价格不是9元？
A: 可能原因：
- 不是新用户
- 选错了配置
- 活动已过期
- 换个配置或活动看看

### Q3: 连接不上服务器？
A: 检查：
- 安全组是否开放22端口
- 密码是否正确
- 服务器是否正在运行
- 等待1-2分钟让服务器完全启动

### Q4: 3000端口无法访问？
A: 必须配置安全组！
- 添加入方向规则：端口3000
- 协议：TCP
- 源地址：0.0.0.0/0

### Q5: 续费多少钱？
A: 第二年续费大约：
- S6型 1核2GB：约100-150元/年
- 可能会有新活动
- 可以重新注册账号享受新用户价
